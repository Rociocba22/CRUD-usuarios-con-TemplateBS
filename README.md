# CRUD USUARIOS

Este proyecto es un crud de usuarios

## Tecnologías
    -Backend: NodeJs
    -Frontend: Javascript

## Instalación

    Ejecutar con `npm run dev`
    Ejecutar en local
```javascript

    - npm install
    - npm run dev

```
